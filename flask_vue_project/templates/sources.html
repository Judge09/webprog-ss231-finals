Website Concept

This website is designed to showcase my skills, projects, and personal interests in a hacker-inspired aesthetic. It is built using Flask for the backend and Vue for dynamic interactivity, ensuring a smooth and responsive user experience. The site features a dynamic feedback system powered by Supabase, an interactive terminal log for displaying logs in a cyberpunk-style interface, and an organized gallery to highlight different aspects of my personal and academic journey. Additional integrations include Bootstrap for styling, FontAwesome for icons, and a matrix-style effect to enhance the immersive feel of the design.
